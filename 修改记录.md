## 🔧 详细修改记录

### 1. 鼠标功能修复

#### 1.1 修复鼠标位置错位问题
**问题：** 鼠标点击位置与实际位置不符

**解决方案：** 实现坐标转换（SessionPage.ets:780-804）
```typescript
// 计算 ImageFit.Contain 的缩放比例
const scaleX = this.screenWidth / this.desktopWidth;
const scaleY = this.screenHeight / this.desktopHeight;
const scale = Math.min(scaleX, scaleY);

// 计算中心对齐偏移量
const displayedWidth = this.desktopWidth * scale;
const displayedHeight = this.desktopHeight * scale;
const offsetX = (this.screenWidth - displayedWidth) / 2;
const offsetY = (this.screenHeight - displayedHeight) / 2;

// 转换触摸坐标到桌面坐标
const desktopX = Math.floor((touchX - offsetX) / scale);
const desktopY = Math.floor((touchY - offsetY) / scale);
```

**提交记录：**
- `1f50c8a` - 修复键盘输入焦点问题和界面拖动问题
- `1113fcc` - 修复键盘焦点导致工具栏弹出问题

#### 1.2 修复 PTR_FLAGS 错误
**问题：** 鼠标点击导致连接断开

**原因：** `PTR_FLAGS_BUTTON1` 使用错误值 `0x0001`

**修复：** 改为正确值 `0x1000`（参考 Golang 实现）

**PTR_FLAGS 定义（LibFreeRDP.ets）：**
```typescript
export const PTR_FLAGS_MOVE = 0x0800;       // 鼠标移动
export const PTR_FLAGS_DOWN = 0x8000;       // 按下
export const PTR_FLAGS_BUTTON1 = 0x1000;    // 左键
export const PTR_FLAGS_BUTTON2 = 0x2000;    // 右键
export const PTR_FLAGS_WHEEL = 0x0200;      // 滚轮
```

#### 1.3 防止界面拖动
**问题：** 鼠标点住时会拖动整个远程桌面界面

**解决方案：**
- 添加 `.hitTestBehavior(HitTestMode.Block)` 阻止事件传播
- 添加 `event.stopPropagation()` 防止事件冒泡

**提交记录：** `1f50c8a`

#### 2.2 键盘事件处理

##### onChange 处理普通字符
```typescript
.onChange((value: string) => {
  if (value.length > this.lastInputLength) {
    // 新字符添加
    const newChars = value.substring(this.lastInputLength);
    for (let i = 0; i < newChars.length; i++) {
      const char = newChars[i];
      const charCode = char.charCodeAt(0);
      // 发送 Unicode 字符
      LibFreeRDP.sendUnicodeKeyEvent(instance, charCode, true);
      LibFreeRDP.sendUnicodeKeyEvent(instance, charCode, false);
    }
  }
})
```

##### onKeyEvent 处理特殊键
```typescript
.onKeyEvent((event: KeyEvent) => {
  const keyCode = event.keyCode;
  const instance = this.session.getInstance();

  // 使用 setTimeout 延迟 key up 事件
  LibFreeRDP.sendKeyEvent(instance, VK_CODE, true);
  setTimeout(() => {
    LibFreeRDP.sendKeyEvent(instance, VK_CODE, false);
  }, 10);
})
```

### 3. 焦点管理和工具栏防误触

#### 3.1 焦点追踪
**问题：** 键盘输入时触发外层 onClick，导致工具栏弹出

**解决方案：** 添加焦点状态追踪
```typescript
@State remoteDesktopHasFocus: boolean = false;
private lastKeyPressTime: number = 0;

// TextInput 焦点事件
.onFocus(() => {
  this.remoteDesktopHasFocus = true;
})
.onBlur(() => {
  this.remoteDesktopHasFocus = false;
})

// 每次按键记录时间
this.lastKeyPressTime = Date.now();
```

#### 3.2 防抖机制
**问题：** 空格键和回车键会触发可访问性的 onClick 事件

**解决方案：** 时间窗口检测（SessionPage.ets:1073-1090）
```typescript
.onClick(() => {
  const now = Date.now();
  const timeSinceLastKey = now - this.lastKeyPressTime;

  // 500ms 内的点击视为键盘触发，忽略
  if (timeSinceLastKey < 500) {
    console.info('忽略键盘触发的点击');
    return;
  }

  // 只在无焦点时切换工具栏
  if (!this.remoteDesktopHasFocus) {
    this.toggleToolbar();
  }
})
```

const initializationOptions: image.InitializationOptions = {

**现象：** 输入时平板输入法会短暂出现

**原因：** TextInput 组件会触发系统输入法

**影响：** 不影响功能，但用户体验稍差

**当前方案：** 防抖时间窗口 500ms

**待验证：** 是否完全解决

**原因：** 可能 keyCode 不匹配
