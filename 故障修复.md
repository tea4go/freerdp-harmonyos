## 问题描述

应用启动后自动连接远程桌面，但连接成功后立即崩溃退出，无法看到远程桌面界面。

### 症状
- ✓ 应用启动后自动连接到 192.168.52.127:3389
- ✓ 连接状态显示"已连接"
- ✓ 应用立即崩溃退出（SIGABRT）
- ✗ 无法看到远程桌面内容

---

## 诊断过程

### 1. 查看崩溃日志

使用 `hdc shell hilog -T FreeRDP` 查看日志,发现关键崩溃信息:

```
Reason:Signal:SIGABRT(SI_TKILL)@0x01317c000000596c from:22892:20020224
Fault thread info:
Tid:23063, Name:.yjsoft.freerdp
#00 pc 00000000001b86d4 /system/lib/ld-musl-aarch64.so.1(raise+216)
#01 pc 0000000000163990 /system/lib/ld-musl-aarch64.so.1(abort+24)
#02 pc 00000000001fee0c /data/storage/el1/bundle/libs/arm64/libfreerdp3.so.3
#03 pc 0000000000200cc8 /.../libfreerdp3.so.3(freerdp_settings_get_uint32+2824)
#04 pc 00000000000103fc /.../libfreerdp_harmonyos.so
#05 pc 00000000002711f4 /.../libfreerdp3.so.3(freerdp_connect+900)
```

### 2. 分析崩溃位置

崩溃发生在 `harmonyos_post_connect` 函数中调用 `freerdp_settings_get_uint32()` 时。

### 3. 根本原因

**核心问题**: `freerdp_settings_get_uint32()` 函数内部触发了 `abort()` 调用。

- FreeRDP 库内部使用 `WINPR_ASSERT` 宏进行参数验证
- 当检测到无效参数时触发 `abort()`,导致应用立即退出
- 这个 abort 发生在 UI 回调 `g_onSettingsChanged` 中调用 `freerdp_settings_get_uint32()` 时

## 解决方案

### 临时修复方案 (已实施)

**策略**: 暂时禁用导致崩溃的 UI 回调调用

**修改文件**: [entry/src/main/cpp/harmonyos_freerdp.cpp](entry/src/main/cpp/harmonyos_freerdp.cpp#L497-L511)

**修改内容**:

```cpp
/* TEMPORARILY DISABLED: These callbacks may cause abort() in freerdp_settings_get_uint32
 * We'll re-enable them once we fix the root cause
 */
LOGW("harmonyos_post_connect: UI callbacks temporarily disabled to prevent crash");

/*
if (g_onSettingsChanged && instance && instance->context && instance->context->settings) {

    UINT32 width = freerdp_settings_get_uint32(settings, FreeRDP_DesktopWidth);
    UINT32 height = freerdp_settings_get_uint32(settings, FreeRDP_DesktopHeight);
    UINT32 bpp = freerdp_settings_get_uint32(settings, FreeRDP_ColorDepth);
    LOGI("harmonyos_post_connect: Settings: %ux%u@%u", width, height, bpp);
    g_onSettingsChanged((int64_t)(uintptr_t)instance, width, height, bpp);
    LOGI("harmonyos_post_connect: g_onSettingsChanged completed");
} else {
    LOGW("harmonyos_post_connect: g_onSettingsChanged callback not set or invalid context");
}
*/

if (g_onConnectionSuccess) {
    LOGI("harmonyos_post_connect: Calling g_onConnectionSuccess");
    g_onConnectionSuccess((int64_t)(uintptr_t)instance);
    LOGI("harmonyos_post_connect: g_onConnectionSuccess completed");
} else {
    LOGW("harmonyos_post_connect: g_onConnectionSuccess callback not set");
}
```

**保留的功能**:
- ✓ 连接建立 (TCP连接成功)
- ✓ GDI 初始化 (图形设备接口)
- ✓ 指针注册 (鼠标/触摸支持)
- ✓ 更新回调设置 (BeginPaint/EndPaint/DesktopResize)
- ✓ 连接成功通知 (`g_onConnectionSuccess`)

**暂时禁用的功能**:
- ✗ `g_onSettingsChanged` 回调 (通知 UI 层桌面尺寸)
